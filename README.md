# LexForge
LexForge is a lightweight, data-driven study environment designed for law students mastering complex legal scenarios. Unlike traditional Q&amp;A tools, LexForge challenges users with high-difficulty "Bar-style" questions stripped of subject-matter spoilers, demanding pure issue-spotting and rigorous analysis.

Built on the ALA Framework (Answer, Legal Basis, Application), this tool bridges the gap between rote memorization and professional legal reasoning. It features a clean JSON-to-HTML pipeline, randomized challenge delivery, and a standardized "LLM Instruction Manual" for generating high-fidelity study sets from any legal material.
